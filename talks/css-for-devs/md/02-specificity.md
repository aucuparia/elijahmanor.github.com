# Pig Latin

1. <!-- .element start="1" value="1" --> If starts with consonant (or consonant cluster), then move the end and append "ay" <!-- .element class="fragment" -->

```
"pig"    -> "igpay"
"banana" -> "ananabay"
"trash"  -> "ashtray"
"happy"  -> "appyhay"
"glove"  -> "oveglay"
```
